Pendahuluan

Ayo ke Masjid adalah Sebuah platform yang dikhususkan untuk mengelola masjid. platform ini sangat simpel dan praktis ketika digunakan oleh pengurus masjid. Platform ini dapat melakukan pencatatan administrasi masjid baik pencatatan zakat, sedekah maupun info mengenai kegiatan masjid.

Pada dokumentasi ini akan dijelaskan bagaimana instalasi dari platfrom Ayo ke masjid lalu akan dlanjutkan penjelasan mengenai fitur-fitur yang ada di paltform ayokemasjid.net

Cara Install Platform #back to top

Software yang dibutuhkan untuk instalasi platform ayokemasjid.net sebagai berikut:

  1. XAMPP (download: XAMPP)
  2. Web Browser (Google Chrome atau Mozilla Firefox)

Setelah melakukan instalasi software yang dibutuhkan, ikutilah langkah-langkah dibawah ini:

XAMPP

Setelah XAMPP terbuka, klik START pada bagian Apache dan MYSQL

Download Platform

Setelah klik START pada bagian Apache dan mysql, lalu masuk ke halaman ayokemasjid.net lalu klik Gitlab Repository

Setelah itu akan dituju halaman gitlab seperti dibawah ini :

Download seluruh file kemudian letakkan file pada lokasi C:\xampp\htdocs\islamic-platform

Import Database

Masuk kehalaman PhpMyadmin

Setelah phpmyadmin terbuka, klik NEW untuk membuat nama database islamic-db

Tanda pembuatan database berhasil adalah terdapat nama databse islamic-db yang telah dibuat pada sisi kiri layar. Langkah selanjutnya adalah import database dari file yang sebelumnya telah didownload. file terdapat pada application -> database

Kembali ke halaman phpmyadmin, lalu klik pada menu import, lalu lakukan import file islamic-db(new).sql

Setelah import berhasil, klik menu structure apabila terlihat seperti gambar dibawah ini, maka import database berhasil.


Setting platform

Setelah melakukan import database, selanjutnya adalah setting platform. langkah pertama yang dilakukan adalah buka browser, lalu ketik localhost/islamic-platform/index.php setelah itu akan muncul form pengisian form profil masjid, lalu isi from tersebut sesuai profil masjid.

Setelah mengisi semua form yang tersedia, kemudian lakukan login sesuai data yang sebelumnya telah diisikan didalam form

Setelah login berhasil, maka akan ditampilkan home awal seperti gambar dibawah ini :

Pada tampilan awal website, akan ditampilkan home awal admin dimana terdapat beberapa informasi seperti total infaq mingguan, total infaq jumat, total jakat fitrah total jamaah masjid, total pengurus masjid dan yang lainnya. admin dapat mengisi informasi tersebut dengan fitur fitur yang ada pada website ini.

Fitur Platform #back to top

Ada beberapa fitur yang tersedia pada platform ayokemasjid ini sebagai berikut:

  1. Data jamaah masjid
  2. Laporan Zakat Fitrah
  3. Laporan Zakat Mal
  4. laporan Infaq
  5. Data Mustahik
  6. Membuat Berita
  7. Membuat list Kegiatan
  8. Data Pengurus Masjid
  9. Galeri Masjid

Setiap fitur akan dijelaskan dibawah ini:

Data Jamaah Masjid

Pada fitur ini, admin dapat menambahkan data diri anggota jama'ah masjid. Dengan adanya fitur ini, pengurus masjid dapat mengetahui jumlah anggota jama'ah masjid, data diri jama'ah, kontak jama'ah, hingga persebaran wilayah jama'ah yang terdafar. Untuk mengakses fitur ini, dapat menuju navbar lalu klik Manajemen -> Jama'ah

Pada halaman jama'ah, klik Tambah jama'ah masjid Setelah itu akan muncul form yang harus diisi untuk calon jama'ah masjid.

Setelah melakukan pengisian, Cek pada halaman awal Data jama'ah, data jama'ah berhasil disimpan


Laporan Zakat Fitrah

Pada fitur ini, admin dapat menambahkan siapa saja yang membayar zakat fitrah, dengan adanya fitur ini, pengelolaan keluar masuknya zakat akan semakin akurat. admin juga dapat mencetak laporan penerimaan zakat yang sebelumnya telah diinputkan. Fitur Pengelolaan zakat fitrah dapat diakses melalui navbar manajemen ->zakat fitrah

Pada halaman zakat fitrah, akan muncul beberapa tombol seperi tambah data zakat fitrah, cetak laporan penerimaan, cetak rekapitulasi saat tombol tambah data zakat fitrah di klik, makan akan muncul form data yang harus diisi, yang nantinya data yang telah diisi akan masuk seperti gambar dibawah ini.

Sedangkan fitur cetak akan mencetak laporan data zakat fitrah kedalam bentuk excel.


Laporan Zakat Mal

Pada fitur ini, admin dapat menambahkan siapa saja yang membayar zakat Mal, dengan adanya fitur ini, pengelolaan keluar masuknya zakat Mal akan semakin akurat. admin juga dapat mencetak laporan penerimaan zakat yang sebelumnya telah diinputkan. Fitur Pengelolaan zakat Mal dapat diakses melalui navbar manajemen ->zakat Mal

Pada halaman zakat fitrah, akan muncul tombol tambah data zakat mal saat tombol tambah data zakat mal di klik, maka akan muncul form data yang harus diisi, yang nantinya data yang telah diisi akan masuk seperti gambar dibawah ini.


Laporan Infaq

Pada fitur ini, admin dapat menambahkan siapa saja yang membayar infaq, dengan adanya fitur ini, pengelolaan masuknya infaq akan semakin akurat. admin juga dapat mencetak laporan penerimaan infaq yang sebelumnya telah diinputkan. Fitur Pengelolaan infaq dapat diakses melalui navbar manajemen ->infaq

Pada halaman infaq, akan muncul tombol tambah data infaq saat tombol tambah data infaq di klik, maka akan muncul form data yang harus diisi, yang nantinya data yang telah diisi akan masuk seperti gambar dibawah ini.


Data Mustahik

Pada fitur ini, admin dapat menambahkan siapa saja yang terdaftar sebagai mustahik, dengan adanya fitur ini, pengelolaan anggota mustahik akan semakin akurat. admin juga dapat mencetak laporan mengenai mustahik yang sebelumnya telah diinputkan. Fitur Pengelolaan mustahik dapat diakses melalui navbar manajemen ->mustahik

Pada halaman mustahik, akan muncul tombol tambah data mustahik saat tombol tambah data mustahik di klik, maka akan muncul form data yang harus diisi mengenai nama mustahik beserta penerimaan zakat yang diterima, yang nantinya data yang telah diisi akan masuk seperti gambar dibawah ini.


Membuat Berita

Pada fitur ini, admin dapat menambahkan berita mengenai masjid maupun membuat sebuah kategori berita. dengan adanya fitur ini, apabila admin ingin menyebarkan suatu informasi maka dapat membuat sebuah berita yang nantinya akan muncul pada halaman website masjid. Fitur Pengelolaan berita dapat diakses melalui navbar manajemen ->Berita terdapat 2 hal yang dapat dilakukan pada fitur berita yaitu membuat list berita dan membuat kategori berita.

Pada gambar diatas adalah halaman kategori berita. admin dapat membuat sebuah kategori berita baru yang nantinya dapat digunakan dalam membuat isi berita. pada halaman kategori berita terdapat tombol tambah kategori berita baru yang mana saat di klik akan muncul form pengisian kategori berita. lakukan pengisian kategori berita lalu simpan, dan kategori berita akan muncul seperti gambar dibawah ini.

Selanjutnya klik pada navbar list berita pada halaman ini admin dapat mengisi suatu bertia serta dapat mengkategorikannya kedalam kategori yang yang sebelumnya telah dibuat

Pada halaman list berita, klik tombol tambah berita baru kemudian isi form yang ada untuk menyimpan berita yang akan di tampilkan pada halaman web. setelah mengisi form tambah berita maka pada halaman list berita akan muncul gambar seperti dibawah ini.

Setelah berita tersimpan, maka berita tersebut akan muncul pada halamn website masjid dan dapat dibaca oleh pengakses website.


Membuat List Kegiatan

Pada fitur ini, admin dapat menambahkan kegiatan yang akan dilakukan oleh masjid contohnya adalah info mengenai kajian rutin. dengan adanya fitur ini, apabila admin ingin menyebarkan suatu informasi kegiatan yangg akan datang maka dapat mengisi form pada fitur kegiatan yang nantinya akan muncul pada website masjid. Fitur Pengelolaan kegiatan dapat diakses melalui navbar manajemen ->Kegiatan->list kegiatan

Pada gambar diatas adalah halaman kategori berita. admin dapat membuat sebuah kategori berita baru yang nantinya dapat digunakan dalam membuat isi berita. pada halaman kategori berita terdapat tombol tambah kategori berita baru yang mana saat di klik akan muncul form pengisian kategori berita. lakukan pengisian kategori berita lalu simpan, dan kategori berita akan muncul seperti gambar dibawah ini.

list kegiatan yang telah dibuat akan muncul pada halaman web masjid dan dapat dibaca oleh seluruh pengunjung web.


Data Pengurus Masjid

Pada fitur ini, admin dapat menambahkan data pengurus masjid baik menjadi ketua, sekretaris, bendahara, maupun anggota. Fitur Pengelolaan kegiatan dapat diakses melalui navbar Pengaturan ->Pengurus Masjid

Pada gambar diatas adalah halaman pengurus masjid. Pada halaman Pengurus masjid terdapat tombol tambah pengurus masjid yang mana saat di klik akan muncul form pengisian pengurus masjid. lakukan pengisian form pengurus simpan, dan kategori berita akan muncul seperti gambar dibawah ini.


Galeri Masjid

Pada fitur ini, admin dapat menambahkan foto mengenai kegiatan yang dilakukan oleh masjid ataupun lainnya. fitur Pengelolaan galeri masjid dapat diakses melalui navbar Pengaturan ->Galeri Masjid

Pada gambar diatas adalah halaman galeri. Pada halaman galeri masjid terdapat form untuk upload gambar ataupun foto kegiatan ang pernah dilakukan beserta deskripsi kegiatannya. Setelah melakukan pengisian form dan upload gambar, sistem akan menyimpan foto kegiatan tersebut dan akan ditampilkan pada halaman web. gambar dibawah adalah kondisi saat foto berhasil disimpan.

Foto yang telah diupload akan masuk kehalaman web dan dapat dilihat atau diakses oleh seluruh pengguna web.
